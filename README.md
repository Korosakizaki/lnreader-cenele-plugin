
# Cenele Plugin for LNReader
Arabic light novel source for LNReader app.
## Source Info
- **Name:** Cenele (فضاء الروايات)
- **URL:** https://cenele.com/
- **Language:** Arabic
- **Type:** Web Novels / Light Novels
## Installation
1. Open LNReader app
2. Go to **Plugins** → **Add Repository**
3. Paste this URL: 
https://raw.githubusercontent.com/Korosakizaki/lnreader-cenele-plugin/main/index.json
4. Install the Cenele plugin

## Repository URL
https://raw.githubusercontent.com/Korosakizaki/lnreader-cenele-plugin/main/index.json

## Features
- Popular novels listing
- Search functionality
- Chapter reading
- Arabic RTL text support
